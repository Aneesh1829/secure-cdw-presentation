Research Paper Presentation

Secure, Verifiable, and Efficient Boolean Keyword Searchable Encryption

for Cloud Data Warehouse

Authors: Somchart Fugkeaw, Lyhour Hak, Thanaruk Theeramunkong

Journal: IEEE Access (2024)

Team: G28

Explore Research

The Challenge

🔒

The Encryption Dilemma

Organizations must encrypt sensitive data (facts + dimensions) before outsourcing to the cloud. However, analytical tools cannot directly query encrypted data, destroying usability.

❌

Existing Limitations

  • No Boolean Query Support (AND, OR, NOT)
  • Inefficient for multi-dimensional data cubes
  • Lack of result verification and integrity
  • Limited role-based access control

Proposed Solution

A hybrid approach combining advanced encryption, multi-level indexing, and blockchain verification.

PHE (Paillier Encryption)

Secures data while enabling computation on encrypted values.

B+Tree

Narrows search space efficiently for range and hierarchical queries.

Inverted Index

Enables lightning-fast keyword matching across dimensions.

Bitmap Index

Executes complex Boolean operations (AND, OR) efficiently.

System Architecture

01

Data Owner

Encrypts multi-dimensional data and uploads to the cloud.

02

Data User

Submits a Boolean query securely via an encrypted trapdoor.

03

Proxy Server

Executes the search using the Hybrid Indexing approach without decrypting.

04

Blockchain

Verifies query validity, user identity, and results integrity.

Security & Performance

🛡️ Security Analysis

  • Confidentiality: Data remains encrypted.
  • Search Privacy: Query hidden using trapdoor.
  • Result Integrity: Verified via Blockchain (SHA-256 hashing).
  • Access Control: Smart contract role restrictions.

⚡ Performance Results

  • Lowest search time compared to existing methods.
  • Search time remains almost constant as data scales up.
  • Handles moderate concurrency extremely efficiently.
  • Evaluated on ~38,000 multi-dimensional records.

Conclusion

"The proposed system successfully balances security, efficiency, and functionality in cloud data warehouse environments."

Enables complex Boolean queries on encrypted data without compromising privacy, while adding trust via Blockchain.
